  <action>
**1. Create `modules/nixos/hyprland.nix`:**

```nix
{ config, pkgs, ... }:
{
  # Hyprland compositor (upstreamed nixpkgs module -- no flake input needed)
  programs.hyprland = {
    enable = true;
    xwayland.enable = true;
    # Do NOT enable UWSM -- conflicts with home-manager systemd integration
    # and has mixed community reception. Skip for Phase 1.
  };

  # Display manager: greetd with tuigreet
  services.greetd = {
    enable = true;
    settings = {
      default_session = {
        command = "${pkgs.greetd.tuigreet}/bin/tuigreet --time --cmd Hyprland";
        user = "greeter";
      };
    };
  };

  # Required for Hyprland session management
  security.polkit.enable = true;

  # XDG portal for screen sharing, file dialogs
  xdg.portal = {
    enable = true;
    extraPortals = [ pkgs.xdg-desktop-portal-hyprland ];
  };
}
```

Critical details:
- Use `programs.hyprland.enable = true` from nixpkgs -- do NOT add a Hyprland flake input.
- greetd command uses `--cmd Hyprland` (capital H) -- this is the session command.
- Do NOT set `programs.hyprland.withUWSM = true` -- UWSM conflicts with `wayland.windowManager.hyprland.systemd.enable` in home-manager and has mixed community reception.

**2. Create `modules/home/hyprland.nix`** -- Hyprland user-level settings:

```nix
{ config, pkgs, ... }:
{
  wayland.windowManager.hyprland = {
    enable = true;
    # systemd.enable defaults to true -- integrates with systemd user session.
    # Do NOT disable unless using UWSM (which we are not).

    settings = {
      # Monitor: auto-detect resolution and position (GPU-agnostic)
      monitor = ",preferred,auto,auto";

      "$mainMod" = "SUPER";
      "$terminal" = "kitty";

      general = {
        gaps_in = 5;
        gaps_out = 10;
        border_size = 2;
        layout = "dwindle";
      };

      decoration = {
        rounding = 8;
      };

      input = {
        kb_layout = "us";
        follow_mouse = 1;
        sensitivity = 0;
      };

      # Minimal keybinds for a working desktop
      bind = [
        "$mainMod, T, exec, $terminal"
        "$mainMod, Q, killactive,"
        "$mainMod, M, exit,"
        "$mainMod, V, togglefloating,"
        "$mainMod, F, fullscreen,"
        # Workspace switching
        "$mainMod, 1, workspace, 1"
        "$mainMod, 2, workspace, 2"
        "$mainMod, 3, workspace, 3"
        "$mainMod, 4, workspace, 4"
        "$mainMod, 5, workspace, 5"
        "$mainMod, 6, workspace, 6"
        "$mainMod, 7, workspace, 7"
        "$mainMod, 8, workspace, 8"
        "$mainMod, 9, workspace, 9"
        # Move window to workspace
        "$mainMod SHIFT, 1, movetoworkspace, 1"
        "$mainMod SHIFT, 2, movetoworkspace, 2"
        "$mainMod SHIFT, 3, movetoworkspace, 3"
        "$mainMod SHIFT, 4, movetoworkspace, 4"
        "$mainMod SHIFT, 5, movetoworkspace, 5"
        "$mainMod SHIFT, 6, movetoworkspace, 6"
        "$mainMod SHIFT, 7, movetoworkspace, 7"
        "$mainMod SHIFT, 8, movetoworkspace, 8"
        "$mainMod SHIFT, 9, movetoworkspace, 9"
        # Window focus
        "$mainMod, left, movefocus, l"
        "$mainMod, right, movefocus, r"
        "$mainMod, up, movefocus, u"
        "$mainMod, down, movefocus, d"
      ];

      # Mouse bindings
      bindm = [
        "$mainMod, mouse:272, movewindow"
        "$mainMod, mouse:273, resizewindow"
      ];
    };
  };
}
```

Key details:
- `monitor = ",preferred,auto,auto"` -- auto-detects any display (VM virtio, AMD, Intel, NVIDIA). The leading comma means "any monitor".
- `$terminal = "kitty"` -- kitty is added to system packages in Task 1.
- `systemd.enable` defaults to `true` in home-manager's Hyprland module -- do NOT explicitly set it to false.
- Do NOT add UWSM-related configuration.

NOTE: The home-manager module wiring in flake.nix is NOT done in this plan. Plan 01-03 will add the `home-manager.nixosModules.home-manager` and `home-manager.users.pilot = import ./modules/home` block to flake.nix. For now, `modules/home/` files exist but are not imported yet.
