<task type="auto">
  <name>Task 2: Provision caelestia wallpaper directory and set Matisse EB font in caelestia.nix</name>
  <files>modules/home/caelestia.nix</files>
  <action>
    Two changes to modules/home/caelestia.nix:

    **A. Caelestia font settings**

    The caelestia shell exposes font configuration via `programs.caelestia.settings.appearance.font.family`. The `sans` key controls the display font used for bar labels, launcher text, and general UI. Set it to MatissePro-EB (which Stylix has already installed system-wide via stylix.nix and fonts.packages):

    ```nix
    settings = {
      bar.status.showBattery = true;
      paths.wallpaperDir = "~/Pictures/wallpapers";
      appearance.font.family = {
        sans = "MatissePro-EB";
        mono = "JetBrainsMono Nerd Font Mono";
      };
    };
    ```

    IMPORTANT: Set `mono` alongside `sans` to lock in JetBrains Mono for caelestia's monospace surfaces (terminal output in OSD, etc.), matching the Stylix monospace declaration. Do NOT set `clock` or `material` — leave those at caelestia defaults (Rubik for clock, Material Symbols Rounded for icons). Overriding `material` would break icon rendering.

    IMPORTANT: `appearance.font.family` is a nested attrset. In Nix, write it as `appearance.font.family = { sans = ...; mono = ...; }` (NOT as dotted keys like `"appearance.font.family.sans" = ...`). The caelestia hm-module serializes settings via `lib.recursiveUpdate` + `builtins.toJSON`, so Nix attrset nesting maps directly to JSON nesting.

    **B. Wallpaper directory provisioning via home.file**

    Caelestia reads wallpapers from `paths.wallpaperDir = "~/Pictures/wallpapers"`. This directory must exist and contain images; caelestia cycles through them. Use `home.file` to provision NGE images from the repo into that directory.

    Add a top-level `home.file` block to modules/home/caelestia.nix (alongside the `programs.caelestia` block):

    ```nix
    home.file = {
      "Pictures/wallpapers/4-21012F64SA53.jpg".source =
        ../../"Neon Genesis Evangelion (Pics)"/"4-21012F64SA53.jpg";
      "Pictures/wallpapers/numbers-interfaces-anime-neon-genesis-evangelion-wallpaper-preview.jpg".source =
        ../../"Neon Genesis Evangelion (Pics)"/"numbers-interfaces-anime-neon-genesis-evangelion-wallpaper-preview.jpg";
    };
    ```

    IMPORTANT: home.file paths are relative to `home.homeDirectory` (/home/pilot), so `"Pictures/wallpapers/..."` expands to `/home/pilot/Pictures/wallpapers/...`. Do NOT use `~/Pictures/wallpapers` as the key — the tilde is not expanded in home.file keys.

    IMPORTANT: The source paths use `../../` to go from `modules/home/` up to the repo root, then into `Neon Genesis Evangelion (Pics)/`. In Nix, directory traversal across path segments with spaces requires quoting each segment separately using the `a/b` path concatenation form shown above. Alternatively, use `builtins.path { path = ../../. + "/Neon Genesis Evangelion (Pics)/4-21012F64SA53.jpg"; name = "4-21012F64SA53.jpg"; }` if the quoted-segment form does not parse cleanly.

    The full updated modules/home/caelestia.nix should look like:

    ```nix
    { ... }:
    {
      # Caelestia shell: unified QuickShell bar + launcher + notifications + lock + sidebar.
      # Replaces the hand-rolled waybar/rofi/mako/hyprlock/hyprpaper/cliphist stack.
      #
      # Theming note: Stylix does NOT target caelestia. Stylix still owns kitty,
      # GTK, Qt, cursor, and the system font stack. Caelestia themes its own QML
      # surfaces via programs.caelestia.settings. Do NOT try to bridge them here;
      # pilot-specific color injection is a later phase (matugen + specialisations).
      programs.caelestia = {
        enable = true;
        systemd = {
          enable = true;
          target = "graphical-session.target";
        };
        cli.enable = true;
        settings = {
          bar.status.showBattery = true;
          paths.wallpaperDir = "~/Pictures/wallpapers";
          appearance.font.family = {
            sans = "MatissePro-EB";
            mono = "JetBrainsMono Nerd Font Mono";
          };
        };
      };

      # Provision NGE wallpapers for caelestia to cycle through.
      # home.file keys are relative to homeDirectory (/home/pilot).
      # Caelestia picks from paths.wallpaperDir above.
      home.file = {
        "Pictures/wallpapers/4-21012F64SA53.jpg".source =
          ../../"Neon Genesis Evangelion (Pics)"/"4-21012F64SA53.jpg";
        "Pictures/wallpapers/numbers-interfaces-neon-genesis-evangelion-preview.jpg".source =
          ../../"Neon Genesis Evangelion (Pics)"/"numbers-interfaces-anime-neon-genesis-evangelion-wallpaper-preview.jpg";
      };
    }
    ```

    NOTE on the `home.file` key for the preview: rename it to a shorter key (removing "anime-" from the destination filename) to avoid overly long paths. The source path still points to the full filename.
