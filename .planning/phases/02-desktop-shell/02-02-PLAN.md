<objective>
Add the second set of desktop-shell tools: Thunar file manager (system service), mako notifications (user), and the hyprlock+hypridle lock pair (user + a system-side PAM hook). This completes the "daily use" surface beyond the bar/launcher/terminal.

Purpose: A desktop without a file manager, notifications, or a working lock screen is unusable. Thunar must be a NixOS module (for gvfs+tumbler system services); mako and hyprlock/hypridle are user-session tools. Critically, hyprlock REQUIRES both a home-manager config AND `programs.hyprlock.enable = true` on the NixOS side -- the latter auto-configures `security.pam.services.hyprlock`. Without it, hyprlock cannot authenticate.
Output: Three new module files (file-manager.nix system-side; mako.nix and lock.nix home-side), wired into aggregators, with PAM correctly set up.
</objective>

  programs.thunar = {
    enable = true;
    plugins = with pkgs.xfce; [
      thunar-archive-plugin
      thunar-volman

Why each line matters (from RESEARCH.md Pitfall 5):
- Without `gvfs`, click-to-mount a USB silently fails and the trash bin does not work.
- Without `tumbler`, thumbnails never render (Thunar shows generic icons).
- Without `thunar-volman`, auto-open dialogs on inserting media do not appear.
- Without `thunar-archive-plugin`, right-click extract/compress is missing (expected file-manager UX).

**Create `modules/home/lock.nix`** (from RESEARCH.md "Pattern 3" + hypridle config). This file declares BOTH hyprlock and hypridle because they are a single user-facing concept (lock now + lock on idle):

```nix
{ ... }:
{
  programs.hyprlock = {
    enable = true;
    settings = {
      general.hide_cursor = true;
      background = [{
        path = "screenshot";
        blur_passes = 3;
      }];
      input-field = [{
        size = "200, 50";
        position = "0, -80";
        placeholder_text = "Pilot authentication";
        fade_on_empty = false;
      }];
      label = [{
        text = "NERV COMMAND";
        position = "0, 160";
        halign = "center";
        valign = "center";
      }];
    };
  };

  services.hypridle = {
    enable = true;
    settings = {
      general = {
        lock_cmd = "pidof hyprlock || hyprlock";
        before_sleep_cmd = "loginctl lock-session";
      };
      listener = [
        { timeout = 300; on-timeout = "loginctl lock-session"; }
        { timeout = 600; on-timeout = "hyprctl dispatch dpms off"; on-resume = "hyprctl dispatch dpms on"; }
      ];
    };
  };
}
```

Notes:
- `background.path = "screenshot"` produces the blurred-last-frame effect. If hyprlock logs "could not grab screen copy" on first test, fall back to `path = "${../../assets/wallpaper.jpg}"` (RESEARCH.md Pitfall 6).
- Do NOT set colors or fonts on the `input-field`/`label`; Stylix themes hyprlock via `stylix.targets.hyprlock`.
- Idle timer is 5 min to lock, 10 min to DPMS off.
